Client¶

Connect to a Duruses server using a Python shell with duruses-client. Connect to a Duruses server in your own code with duruses.client.Client.

In this section you’ll learn how to connect using the command-line or in your own code, and you’ll see some basic examples of how to use databases once connected.

For more complete information, see API Reference.

Connect using the command-line¶

Run duruses-client, which connects to a Duruses server, then provides a Python shell giving you access to useful objects:

$ duruses-client
Duruses shell
  Connection - duruses.connection.Connection class
  client - duruses.client.Client instance
>>>

Note

By default, duruses-client connects to 127.0.0.1 on port 22972. Connect to a different host and/or port by using --host and --port, as in this example:

$ duruses-client --host=11.22.33.44 --port=3333

Note

If you have IPython installed, duruses-client will favor that over the default Python shell:

$ duruses-client
Duruses shell
  Connection - duruses.connection.Connection class
  client - duruses.client.Client instance
In [1]:

In the shell, use the client object’s storage() method to get a durus.client.ClientStorage instance for a named database. Once you have that, create a Connection instance:

>>> conn = Connection(client.storage('userdata'))

With the Connection instance, use the standard Durus API to operate on the database:

>>> root = conn.get_root()
>>> root['users'] = {}
>>> root['groups'] = {}
>>> conn.commit()

Connect in your own code¶

Create a Client instance, then perform operations on it as above in Connect using the command-line:

from durus.connection import Connection
from duruses.client import Client

# Connect to default server.
client = Client()

# Work with the "userdata" database.
conn = Connection(client.storage('userdata'))

# Create some objects and commit them in a transaction.
root = conn.get_root()
root['users'] = {}
root['groups'] = {}
conn.commit()
