Requirements¶

  • Python 2.6+

  • setuptools

    If you do not have setuptools, follow the setuptools installation instructions.

Installation¶

Use easy_install to install the xdserver package and its dependencies:

$ easy_install xdserver

If any dependencies do not install, such as in the case of using the Windows platform without a C compiler, use our package index:

$ easy_install -i http://schevo.org/eggs/index xdserver
