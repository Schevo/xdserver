Welcome to Duruses’ documentation¶

Author:Matthew Scott <matt@11craft.com>
Date:September 18, 2009
Description:Durus Extended Server

Durus is an object database for Python. It allows multiple clients to operate on a single database via a wire protocol, and allows one process at a time to directly access a database file.

Duruses is an extended server for Durus providing these enhancements:

  • Serve multiple databases with one server process. Operate on multiple databases with one connection.
  • Asynchronous server, using cogen to offer platform-specific performance enhancements.
