Development¶

Getting the latest code¶

Using git, clone the xdserver repository:

$ git clone git://github.com/11craft/xdserver.git

You can view source code online at the xdserver github repository page.

Contributing code¶

Our preferred way of receiving changes is for you to fork the project and submit a pull request on GitHub.
