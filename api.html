API Reference¶

duruses.client¶

class duruses.client.Client(host='127.0.0.1', port=22972)¶

Connects to a duruses server.

Parameters:
  • host (string) – Host name or IP address of server to connect to.
  • port (integer) – Port server is listening on.
close(db_name)¶

Close the named database.

Parameter:db_name – Name of database to close.
Ptype db_name:string
destroy(db_name)¶

Destroy the named database.

Parameter:db_name – Name of database to destroy.
Ptype db_name:string

Note

The database will not be destroyed if it is currently open.

Warning

This will permanently delete the file containing the database on the server.

disconnect()¶
Disconnect from the server.
list_all()¶

List all databases available on the server.

Return type:list of strings
list_open()¶

List open databases on server.

Return type:List of strings
open(db_name)¶

Open the named database.

Parameter:db_name – Name of database to open.
Ptype db_name:string
quit()¶

Shut down the server process and disconnect.

Todo

Determine if it is necessary to keep “quit” in the protocol or if a server should only be shut down by process control on the system that the server is running on.

server_protocol()¶

Get the protocol used by the server.

Return type:4-byte string
storage(db_name)¶

Return a Durus storage object for the named database.

The database is opened on the server if it was not open already.

Parameter:db_name – Name of database to return storage object for.
Ptype db_name:string
Return type:ClientStorage
class duruses.client.ClientStorage(client, db_name)¶

Durus storage for a Duruses database.

Follows the durus.storage.Storage API.

Please use storage() to create storage instances, rather than creating instances of this class directly.

duruses.server¶

class duruses.server.Server(scheduler, path, storage_class=<class 'durus.file_storage.FileStorage'>, host='127.0.0.1', port=22972)¶

Provides access to databases for duruses clients.

In most cases your code will not use this class directly. Use duruses-server instead; see Server.
